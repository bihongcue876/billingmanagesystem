# 计费管理系统 - 使用说明

> 详细的功能操作指南

---

## 快速开始

### 第一次使用

1. 运行 `app.exe`
2. 登录界面输入 `1` 进入登录
3. 输入默认管理员账号：`admin`
4. 输入默认密码：`123456`
5. 登录成功后进入主菜单

### 登录界面
```
---------------------------
     计费管理系统
---------------------------
  1. 登录
  0. 退出系统
---------------------------
请输入选项：
```

## 管理员管理

### 修改密码
1. 主菜单选择 `6` 进入管理员管理
2. 选择 `1` 修改密码
3. 输入当前密码（隐藏输入）
4. 输入新密码（6-12 位，隐藏输入）
5. 确认新密码（隐藏输入）

### 添加管理员
1. 主菜单选择 `6` 进入管理员管理
2. 选择 `2` 添加管理员
3. 输入新管理员账号（6-12 位）
4. 输入密码（6-12 位，隐藏输入）
5. 确认密码（隐藏输入）

### 删除管理员
1. 主菜单选择 `6` 进入管理员管理
2. 选择 `3` 删除管理员
3. 输入要删除的管理员账号
4. 确认删除（输入 `y`）

> **注意**：不能删除自己

### 管理员列表
1. 主菜单选择 `6` 进入管理员管理
2. 选择 `4` 管理员列表
3. 查看所有管理员账号

## 账户管理

### 查询账户
1. 主菜单选择 `1` 进入账户服务
2. 选择 `1` 查询账户
3. 输入卡号
4. 查看账户信息（状态、余额、使用次数等）

### 账户列表
1. 主菜单选择 `1` 进入账户服务
2. 选择 `5` 账户列表
3. 查看所有有效账户
4. 显示总账户数和总余额
5. 可选择导出到文件（输入 `y`）

> **文件导出**：导出为 TXT 文件，保存至 `output` 目录

### 注册账户（开户）
1. 主菜单选择 `1` 进入账户服务
2. 选择 `3` 注册账户
3. 输入卡号（最多 18 位，只能包含字母、数字、下划线）
4. 输入密码（最多 8 位，隐藏输入）
5. 输入开卡金额（大于 0 且不超过 1000 元）
6. 注册成功，自动记录充值交易

### 账户信息变更
1. 主菜单选择 `1` 进入账户服务
2. 选择 `2` 账户信息变更
3. 输入卡号
4. 选择修改项目（1-密码，2-状态）
5. 输入新值

### 删除账户（注销）
1. 主菜单选择 `1` 进入账户服务
2. 选择 `4` 删除账户
3. 输入卡号
4. 系统根据余额情况处理：
   - **有余额（>0）**：提示退费或确认后退还余额
   - **欠费（-100 ≤ 余额 < 0）**：拒绝注销，要求先还债
   - **欠费过多（<-100）**：允许强制注销（可能是系统错误）
5. 确认注销（输入 `y`）

> **注意**：注销为软删除，保留记录

## 上下机管理

### 用户上机
1. 主菜单选择 `2` 进入上下机服务
2. 选择 `1` 用户上机
3. 输入卡号
4. 输入密码（最多 5 次机会，隐藏输入）
5. 选择计费套餐（直接回车使用默认套餐）
6. 上机成功

### 用户下机
1. 主菜单选择 `2` 进入上下机服务
2. 选择 `2` 用户下机
3. 输入卡号
4. 系统自动计算费用（根据套餐类型：简单计费或分段计费）
5. 下机成功（如欠费会提示）

## 计费标准设定

### 新增标准（支持分段计费）
1. 主菜单选择 `3` 进入计费标准设定
2. 选择 `1` 新增标准
3. 输入套餐编号
4. 输入套餐名称
5. 选择计费单位（0-分钟，1-小时）
6. 选择计费模式：
   - **简单计费**：输入单价
   - **分段计费**：
     a. 输入分段数量（1-10）
     b. 依次输入每段的开始时间、结束时间（-1表示无上限）、单价
     c. 例如：0,1,5 表示0-1小时5元/小时

### 查询标准
1. 主菜单选择 `3` 进入计费标准设定
2. 选择 `2` 查询标准
3. 输入套餐编号（直接回车查询所有）
4. 查看套餐信息
5. 分段计费会显示每段详情

> **文件导出**：导出为 CSV 文件，保存至 `output` 目录

### 修改标准
1. 主菜单选择 `3` 进入计费标准设定
2. 选择 `3` 修改标准
3. 输入套餐编号
4. 选择修改项目（1-计费单位，2-单价）
5. 输入新值

### 删除标准
1. 主菜单选择 `3` 进入计费标准设定
2. 选择 `4` 删除标准
3. 输入套餐编号
4. 确认删除

> **注意**：删除为软删除

## 计费规则

### 简单计费模式

#### 分钟计费
| 上网时长 | 取整规则 |
|----------|----------|
| < 15 分钟 | 向下取整 |
| 15-60 分钟 | 向上取整 |
| ≥ 60 分钟 | 向下取整 |

#### 小时计费
| 上网时长 | 取整规则 |
|----------|----------|
| 全部 | 向上取整 |

### 分段计费模式

可设置多个时间段，每段有不同的单价：
- 每段费用 = 该段计费时长 × 该段单价
- 每段计费时长向上取整
- 总费用 = 各段费用之和

**示例**：
```
分段设置：
第1段：0-1小时，单价5元/小时
第2段：1-2小时，单价3元/小时  
第3段：2小时以上(-1)，单价2元/小时

上机1.5小时费用计算：
- 第1段：1小时 × 5元 = 5元
- 第2段：0.5小时向上取整=1小时 × 3元 = 3元
- 合计：8元
```

## 文件导出功能

所有查询结果支持导出到文件：

| 功能 | 导出格式 | 文件名示例 |
|------|----------|------------|
| 账户列表 | TXT | accounts_20260417_143022.txt |
| 计费标准 | CSV | billing_standards_20260417_143022.csv |
| 消费记录 | CSV | consumption_history_20260417_143022.csv |
| 营业额日志 | CSV | totalsales_20260417_143022.csv |
| 上下机日志 | CSV | uselogs_20260417_143022.csv |

- 导出目录：`output/`
- 文件名格式：`功能名_YYYYMMDD_HHMMSS.扩展名`
- 程序会自动创建 `output` 目录

## 常见问题

### Q: 忘记密码怎么办？
**A**: 使用其他管理员账号登录，在管理员管理中修改密码。

### Q: 账户注销后能恢复吗？
**A**: 可以。重新注册相同卡号，系统会自动恢复并初始化信息。

### Q: 卡号有什么格式要求？
**A**: 卡号只能包含字母、数字、下划线，长度不超过 18 位。

### Q: 如何查看某个账户的所有交易记录？
**A**: 在财务系统中选择"账户消费记录"，或选择"按日期查询"并输入卡号。

### Q: 如何使用分段计费？
**A**: 在计费标准设定中选择"新增标准"，选择"分段计费"模式，配置各时间段和单价。

### Q: 上机后套餐被删除了怎么办？
**A**: 系统会使用默认费率（0.1 元/分钟）计算。

### Q: 如何备份数据？
**A**: 复制 `data/` 文件夹到安全位置即可。建议定期备份。

### Q: 日期查询的格式是什么？
**A**: 日期格式为 `YYYY-MM-DD`，例如 `2026-03-27`。

### Q: 充值/退费多少算大额？
**A**: 超过 500 元的充值或退费会被视为大额操作，需要二次确认。

## 注意事项

1. **首次运行** - 自动创建 `data/` 目录和数据库文件，并创建默认管理员账号（admin/123456）
2. **密码安全** - 所有密码输入均显示 `*` 号，支持退格键删除
3. **账号注销** - 采用软删除机制，根据余额情况处理（见上文）
4. **数据库路径** - 所有数据库文件位于 `data/` 目录下，请勿手动修改
5. **日志分表** - 上下机日志按年份自动分表（如 `loginout2026`）
6. **登录失败** - 管理员登录失败 5 次后返回登录界面
7. **卡号格式** - 只能包含字母、数字、下划线，长度不超过 18 位
8. **金额限制** - 开卡金额限制在大于 0 且不超过 1000 元之间
9. **大额确认** - 充值/退费超过 500 元时会要求二次确认
10. **删除确认** - 删除账户时会显示账户信息并要求确认
11. **数据备份** - 定期备份 `data/` 文件夹以防数据丢失
12. **日期查询** - 日期格式为 `YYYY-MM-DD`
13. **文件导出** - 查询结果可导出到 `output` 目录
14. **金额精度** - 所有金额精确到分，计算时四舍五入
